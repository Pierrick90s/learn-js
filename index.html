Apprendre les bases de JavaScript

  1. Vous devez accéder à la console du devTools (touche F12 de votre clavier) de votre navigateur pour afficher les résultats.
  2. Vous pouvez retrouver les leçons et les exercices dans le dossier "js". À chaque fin de leçons, vous avez une série de cinq à réaliser.
  3. Vous pouvez utiliser le fichier "test.js" à la racine de ce dossier pour tester votre code pendant les exercices.
  4. Le dossier "data" vous servira pour stocker les fichiers .json que vous allez créer pendant les exercices pratiques.
